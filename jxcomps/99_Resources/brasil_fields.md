# 📦 Dependência: Brasil Fields

## 📝 Descrição
Conjunto de formatadores e validadores específicos para o padrão brasileiro (CPF, CNPJ, CEP, Moeda).

## 🛠️ Uso no JxComps
* **Campos de Texto:** Essencial para o funcionamento do [[JxTextField]].
* **Validação:** Aplicado nos modelos de dados do [[jxrdata]].
* **Formatadores:** CPF, CNPJ, CEP e Telefone.

## 🔗 Relacionados
* **Componente:** [[JxTextField]]
* **Módulo:** [[jxrdata]]
